# Leap Vue.js Webapp

## Install

Make sure you have Node.js Installed

Install Vue globally
```
npm i -g @vue/cli
```

Install dependencies and copy .env file
```
npm install
cp .env.template .env
```

Run live server
```
npm run serve
```
